Was sind Registry Tweaks?

Die Windows-Registry ist das zentrale Herz des Betriebssystems. Hier speichert Windows Einstellungen für Programme, Hardware und das System selbst.

Registry Tweaks ändern diese Einstellungen direkt, um Leistung, FPS oder Reaktionszeit zu verbessern.

⚠️ Hinweis: Falsche Änderungen können Instabilität verursachen. Immer vorher ein Backup erstellen oder die Registrierung exportieren.

1. Netzwerk-Optimierungen

  • NetworkThrottlingIndex deaktivieren
    Zweck: Verhindert künstliche Bandbreiten-Limits.
    Schritte:
    1. Windows-Taste + R → regedit → OK
    2. Zum Pfad navigieren: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile
    3. Den Wert NetworkThrottlingIndex finden und auf ffffffff setzen (HEX)
    4. PC neu starten
  • TCP Ack Delay reduzieren
    Zweck: Verkürzt Latenz beim Senden von Paketen.
    Schritte:
    1. Regedit öffnen
    2. Zum Pfad navigieren: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters\Interfaces\[Adapter]
    3. Neuen DWORD-Wert TcpAckFrequency erstellen → Wert = 1
    4. Neustart
  • Nagle Algorithmus deaktivieren
    Zweck: Reduziert Input Lag in Shootern.
    Schritte:
    1. Regedit öffnen
    2. Pfad: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters\Interfaces\[Adapter]
    3. DWord-Werte TcpNoDelay
    4. Neustart

Ergebnis: stabilere Verbindung, weniger Mikroruckler in Online-Spielen.

2. Input & Responsiveness

  • MMCSS Optimierungen
    Zweck: Priorität für Spiele-Threads erhöhen.
    Schritte:
    1. Regedit öffnen
    2. Pfad: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile
    3. Den Wert TaskschedulerPriority
    4. Neustart
  • SystemResponsiveness
    Zweck: Hintergrundprozesse blockieren weniger CPU.
    Schritte:
    1. Regedit öffnen
    2. Pfad: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile
    3. DWord SystemResponsiveness
    4. Neustart
  • Mouse Acceleration deaktivieren
    Zweck: Präzisere Mausbewegungen in Shootern.
    Schritte:
    1. Regedit öffnen
    2. Pfad: HKEY_CURRENT_USER\Control Panel\Mouse
    3. Werte MouseSpeedMouseThreshold1MouseThreshold2
    4. Abmelden / Neustart

3. Game DVR & Hintergrunddienste

Windows aktiviert standardmäßig Game DVR und Xbox-Services, die FPS kosten können.

  • Game DVR deaktivieren
    Pfad: HKEY_CURRENT_USER\System\GameConfigStore
    DWORD GameDVR_Enabled auf 0
  • Xbox Game Bar deaktivieren
    Pfad: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\GameDVR
    DWORD AppCaptureEnabled auf 0
  • Hintergrunddienste pausieren
    Empfehlung: Dienste wie Xbox Networking, Cortana bei Spielen deaktivieren über services.msc

Diese Tweaks wirken sich besonders auf ältere oder mittelstarke Systeme positiv aus.

Für wen sind Registry Tweaks geeignet?

  • ✔ Spieler, die jede Millisekunde FPS rausholen wollen
  • ✔ Nutzer, die sich trauen, Registry zu bearbeiten
  • ❌ Anfänger ohne Backup: bitte vorsichtig

Registry Tweaks sollten erst durchgeführt werden, **nachdem Grundlagen und Pro Tweaks** erledigt wurden.
